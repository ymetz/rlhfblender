# RLHF-Blender

Implementation for the backend RLHF-Blender: A Configurable Interactive Interface for Learning from Diverse Human Feedback

You can find the user interface is a separate repository [RLHF-Blender-UI](https://github.com/ymetz/rlhfblender-ui). However, this repository containts everything you need to install the full project.
